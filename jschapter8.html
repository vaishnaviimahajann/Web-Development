Array methods

  • for each : arr.foreach(somefunction)or name
  • map : letnewarr =arr.map(somefunction)
  • filter : let newarr = arr.filter(somefunction)
  • every :returns true if every elemet of array gives true for some function .else return false arr.every(somefunction)
  • some : returns true if any element of array gives true for some function .else return false arr.some(somefunction)

  • reduce : it reduces the array to single value arr.reduce(function(accumulator,currentvalue,index,arr){..},initialvalue)

spread : expand an iterable into multiple values

function funcname(...args){..}

destructuring : extracting values from an array or object
