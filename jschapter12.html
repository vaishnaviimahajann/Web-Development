Async Function

Async functions are a way to write asynchronous code in JavaScript that looks synchronous. They return a Promise and can be used with the await keyword to pause execution until a Promise is resolved.

async and await keywords are used to write asynchronous code in a more readable and manageable way.

await is used inside an async function to pause execution until a Promise is resolved.

API :Application Programming Interface

An API (Application Programming Interface) is a set of rules and protocols that allows different software applications to communicate with each other. It defines how requests and responses should be structured, enabling developers to interact with external services and data.

APIs can be used to access data, perform operations, and integrate third-party services into applications. They are essential for building modern web applications and enabling interoperability between different systems.

JSON : Java Script Object Notation

JSON (JavaScript Object Notation) is a lightweight data interchange format that is easy for humans to read and write, and easy for machines to parse and generate. It is commonly used for transmitting data between a server and a web application as an alternative to XML.

JSON is based on a subset of the JavaScript programming language and uses a simple syntax

Testing API Request

To test API requests, you can use tools like Postman, cURL, or browser developer tools. These tools allow you to send HTTP requests to an API endpoint and view the responses, making it easier to debug and understand how the API works.

When testing API requests, you can specify the HTTP method (GET, POST, PUT, DELETE, etc.), set headers, and include a request body if needed. The response from the API will typically include a status code, headers, and a response body containing the data or message returned by the API.
