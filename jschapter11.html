Call Stack

Call Stack is a mechanism that tracks function calls in JavaScript. It keeps track of the execution context of each function call, allowing JavaScript to know which function is currently running and what functions are waiting to be executed.

breakpoints

Breakpoints are points in the code where execution can be paused for debugging purposes.

Callback hell refers to a situation where multiple nested callbacks are used, making the code difficult to read and maintain. It often occurs when dealing with asynchronous operations.

Promises

Promises are a way to handle asynchronous operations in JavaScript. They represent a value that may be available now, in the future, or never. Promises can be in one of three states: pending, fulfilled, or rejected.

Promises provide a cleaner and more manageable way to handle asynchronous code compared to callbacks, helping to avoid callback hell.
